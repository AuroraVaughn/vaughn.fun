Projects

By heart is a single page web application to help people memorize passages of text and speeches. The premise is that by removing letters from a passage of text it forces users to focus and increases the cognitive load, which helps retention. There are tools such as progressive hints when you mouse over a word and time line by line practice.

By heart was created with React, Redux, Sequelize, Postgres, and Semantic UI.

GitRunner is an aspirational project built by me during a four and half day hackathon at Fullstack Academy. The program recursively collects the commits from a GitHub repo and builds a tree data structure. It then uses autonomous agents to interpret the tree into a JSON map object that PhaserJS can use. The aspirational goal is a game that allows developers to get to know the shape of the file structure by playing a game that mimics it.
